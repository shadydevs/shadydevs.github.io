Dynamic landing page

This project is part of the Udacity Front End program. The HTML and CSS templates were provided as a template by Udacity and modified by me. I wrote the JavaScript code to manipulate the DOM to add a dynamically adjusting navigation bar.

When a section element is added to the HTML, a new navbar link is added automatically with the data-nav value as the name. When the link is clicked, the page scrolls to the appropriate section.
